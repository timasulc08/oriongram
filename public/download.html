Чтобы разместить файлы для скачивания на Firebase Hosting:
  1. Положи файлы в public/ (например public/app-debug.apk, public/OrionGram-Setup.exe)
  2. Собери и задеплой: npm run build → firebase deploy --only hosting
Если файлы большие, лучше хранить их на GitHub Releases или Google Drive и вставить сюда ссылки.
